# Bot-Discord-V2

## Setup project

### add `.env` file

insert 4 value :

```env
# Discord Server
TOKEN="x" # Discord Token
CLIENT="x" # Discord Client ID
GUILD="x" # Discord Guild ID
```

## Install dependencies

```bash
npm install
yarn install
pnpm install
```

## Run project

First console :

```bash
npm run buildCommand
yarn buildCommand
pnpm run buildCommand
```

Second console :

```bash
npm run startBot
yarn startBot
pnpm run startBot
```
