## Setup project

### add `.env` file

insert 4 value :

```env
# Discord Server
TOKEN="x" # Discord Token
CLIENT="x" # Discord Client ID
GUILD="x" # Discord Guild ID
```

## Install dependencies

```bash
npm install
yarn install
pnpm install
```

## Run project


To register you bot's commands:
```bash
npm run build
yarn build
pnpm run build
```

To launch the bot (W.I.P):
```bash
npm run dev
yarn dev
pnpm run dev
```

To launch the bot (LIVE):

*(will update from github before launch and thus erase any local modification)*
```bash
npm run start
yarn start
pnpm run start
```
